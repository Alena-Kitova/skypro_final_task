Git

  • git init - Создание нового репозитория Git в пустой директории или превращение существующей директории в новый репозиторий Git.
  • git clone - Клонирование репозитория из удаленного хранилища на локальную машину.
  • git add - Добавление отдельных файлов или всех файлов в область подготовленных файлов.
  • git status - Проверка статуса репозитория.
  • git commit -m - Внесение изменений однострочным сообщением или через редактор.
  • git log -p - Просмотр истории коммитов с изменениями.
  • git branch - Создание ветки.
  • git checkout - Переход на ветку.
  • git push - Внесем изменения на удаленный репозиторий.
  • git pull - Подтянет изменения из удаленного репозитория в локальный.
